# challenge-maker

#### Sobre: 
Challenge desenvolvido usando ReactJs!

## 

#### Tecnologias:
Frontend: React, CSS Modules (com sass)<br />
Backend: Json-Server

##

#### Instalação:
> cd backend <br />
> npx json-server --watch db.json <br />
> cd .. <br />
> cd frontend <br />
> npm install <br />
> npm start <br />
